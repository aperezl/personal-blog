# Parte 1

Inicializar git

```shell
git init
```

Crear nuevo proyecto

```shell
npx create-next-app .
```

Vincular repositorio

```
git commit -m "create next project"
git remote add origin git@github.com:aperezl/personal-blog.git
git branch -M main
git push -u origin main
```

Instalar dependencias

```
npm i tailwindcss@latest
npm i postcss@latest
npm i autoprefixer@latest
npm i @tailwindcss/typography
```

Crear archivo de configuración para next

```shell
npx tailwindcss init
```

Pasar el fichero `tailwind.config.js` de

```javascript
module.exports = {
  purge: [],
  darkMode: false, // or 'media' or 'class'
  theme: {
    extend: {},
  },
  variants: {
    extend: {},
  },
  plugins: [],
};
```

a:

```javascript
module.exports = {
  purge: ["./pages/**/*.{js,mdx}", "./components/**/*.js"],
  darkMode: false, // or 'media' or 'class'
  theme: {
    extend: {},
  },
  variants: {
    extend: {},
  },
  plugins: [require("@tailwindcss/typography")],
};
```

crear el fichero `postcss.config.js`

```javascript
module.exports = {
  plugins: ["tailwindcss"],
};
```

Añadir al fichero `global.css`

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
